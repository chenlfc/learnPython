* 修改字符串的大小写

      str.title() 将字符串输出为首字母大写

* 合并（拼接）字符串
  * 使用加符号+连接2个字符串
  * 删除空白

            str.rstrip()  # 删除右侧空白
            str.strip()   # 删除头尾处所有空白

* ### 模块 datetime
  * 将字符串'2014-7-1'转换为一个表示相应时间的对象

        >>> from datetime import datetime # 导入时间模块
        >>> first_date = datetime.striptime('2014-7-1', '%Y-%m-%d') # 调用方法将字符串转换为时间对象
        >>> print(first_date) # 打印输出时间对象内容
        2014-07-01 00:00:00

  * strptime()可接受的各种实参，如下表

    |实参|含义|
    |:---------:|:-|
    |%A|星期的名称，如Monday|
    |%B|月份名，如January|
    |%m|用数字表示的月份（01~12）|
    |%d|用数字表示月份中的一天（01~31）|
    |%Y|四位的年份，如2015|
    |%y|两位的年份，如15|
    |%H|24小时制的小时数（00~23）|
    |%I|12小时制的小时数（01~12）|
    |%p|am或pm|
    |%M|分钟数（00~59）|
    |%s|秒数（00~61）|
